# Vue-SimpleApp
This is simple Vue app with use of fake API. \
Used: `vuex` `vue-router` `typescript`

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
